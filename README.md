## 📊 Featured Projects

### 🎮 Steam Games Market Analysis

**Goal:** Understand how pricing, content, and user behavior affect game success

**Key Highlights:**

* Analyzed ~90,000 games (sampled ~7,000)
* Built KPI framework for engagement & satisfaction
* Identified:

  * Free-to-play → higher engagement
  * Paid → higher quality reviews
* Created interactive dashboard in Google Sheets

🔗 Project Repo:
https://github.com/mahir-m01/SectionC_Group2_SteamGames

---

### 🚕 Uber & Ola Ride Analytics

**Goal:** Analyze ride cancellations and optimize operations

**Key Highlights:**

* Worked on ~100,000 ride records
* Identified:

  * Driver inactivity as major cancellation factor
  * High-risk pickup locations
  * Peak-hour inefficiencies
* Designed dashboard for operational insights

🔗 Project Repo:
https://github.com/Rudra-codee/SectionC_Group04_Uber-Ola-Ride-Analytics
